# 🚀 Vokabeltrainer – Schnellstart

## ⚡ Installation (5 Minuten)

### Auf Synology Web Station

1. **Dateien herunterladen**
   - `index.html`, `styles.css`, `app.js`, `ocr.js`

2. **Auf Synology hochladen**
   ```
   File Station → /web → Neuer Ordner "vokabeltrainer"
   → Alle 4 Dateien hochladen
   ```

3. **App öffnen**
   ```
   http://192.168.1.100/vokabeltrainer/
   ```

### Lokal testen

- Öffne `index.html` im Browser
- Fertig! ✅

---

## 📖 Erste Schritte (10 Minuten)

### 1. Vokabel hinzufügen

```
Klick: Vokabeln
Klick: ➕ Neue Vokabel
Eingabe:
  Deutsch: "Haus"
  Englisch: "house"
  Beispiel: "The house is big." (optional)
Klick: Speichern
```

### 2. Liste erstellen

```
Klick: Listen
Klick: ➕ Neue Liste
Eingabe:
  Name: "Unit 1"
  Beschreibung: "Grundwortschatz" (optional)
Klick: Speichern
```

### 3. Lernen starten

```
Klick: Lernen
Wähle: Lernmodus (Karteikarten / Quiz / Schreiben)
Wähle: Liste (optional)
Klick: Lernen starten
```

### 4. Fortschritt verfolgen

```
Klick: Fortschritt
Sieh: Statistiken, Diagramme, Aktivitäten
```

## 🎓 Lernmodi erklärt

### 🎴 Karteikarten
- Sieh deutsche Vokabel
- Klick "Umdrehen" für englische Übersetzung
- Bewerte: ✅ Gewusst oder ❌ Nicht gewusst

### ❓ Quiz
- Sieh deutsche Vokabel
- Wähle richtige Übersetzung aus 4 Optionen
- App zeigt sofort, ob richtig

### ⌨️ Schreiben
- Sieh deutsche Vokabel
- Tippe englische Übersetzung
- Klick "Überprüfen"

---

## 📸 OCR-Funktion (Foto-Import)

### So funktioniert's

1. Fotografiere Vokabelliste aus Schulbuch
2. Klick: **Vokabeln** → **📸 Aus Foto importieren**
3. Lade Foto hoch
4. Klick: **Foto analysieren**
5. Überprüfe erkannte Vokabeln
6. Klick: **Alle importieren**

### Tipps für beste Ergebnisse

- ✅ Gute Beleuchtung
- ✅ Klare, lesbare Schrift
- ✅ Gerade Foto (nicht schräg)
- ✅ Format: "Deutsch - Englisch"
- ❌ Keine Schatten oder Reflexionen

---

## 💾 Daten sichern

### Backup erstellen

```
Klick: Fortschritt
Klick: 📥 Daten exportieren
→ JSON-Datei wird heruntergeladen
→ Speichere sicher
```

### Backup wiederherstellen

```
Klick: Fortschritt
Klick: 📤 Daten importieren
→ Wähle JSON-Datei
→ Daten wiederhergestellt
```

## ❓ Häufige Fragen

**F: Funktioniert die App offline?**
A: Teilweise. Nach dem ersten Laden funktioniert die App ohne Internetverbindung, aber OCR und Diagramme brauchen Internet.

**F: Wo werden meine Daten gespeichert?**
A: Im Browser LocalStorage auf deinem Gerät. Nicht auf einem Server.

**F: Was passiert, wenn ich den Browser-Cache leere?**
A: Deine Daten gehen verloren. Exportiere regelmäßig ein Backup!

**F: Kann ich die App auf mehreren Geräten nutzen?**
A: Ja, aber die Daten sind auf jedem Gerät separat. Exportiere/importiere Backups um zu synchronisieren.

**F: Welche Browser funktionieren?**
A: Chrome, Firefox, Safari, Edge – alle modernen Browser.

## 🎯 Lern-Tipps

1. **Täglich 15 Minuten** ist besser als 1x pro Woche 2 Stunden
2. **Verschiedene Modi** nutzen – Abwechslung hilft beim Lernen
3. **Listen nutzen** – Organisiere nach Kapiteln
4. **Fortschritt verfolgen** – Motivation durch Erfolge
5. **Regelmäßig sichern** – Exportiere deine Daten wöchentlich

## 🔧 Fehlerbehebung

| Problem | Lösung |
|---------|--------|
| App lädt nicht | URL überprüfen, Seite aktualisieren (Strg+F5) |
| OCR funktioniert nicht | Internet überprüfen, anderes Foto versuchen |
| Daten sind weg | Browser-Cache geleert? Backup importieren? |
| Sehr langsam | Internet überprüfen, Browser neu starten |
